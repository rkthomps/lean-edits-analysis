# Adding ability to click into edits to see before and after

## Description
Right now, a session view shows a heatmap of edits for both files and declarations.
I want to be able to dive deeper into these visualizations.  
In particular, for each FileChangeEvent or DeclChangeEvent, 
I want to be able to click on the event and see the state of the file before and after
the change (focused on the part of the file where the change occured)
